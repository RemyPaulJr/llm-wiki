# Linux Hostname

**Summary**: Procedures for viewing and modifying the system hostname on Linux.
**Tags**: #linux #system-admin #hostname
**Created**: 2026-04-13

---
## Content

### View Hostname
Display the current hostname:
```bash
hostname
```

### Change Hostname
Change the machine's hostname permanently:
```bash
sudo hostnamectl set-hostname <name>
```
